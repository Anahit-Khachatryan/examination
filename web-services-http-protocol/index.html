Web Services-ները սերվերներ են, որոնք թույլ են տալիս մեքենաներին հաղորդակցվել ինտերնետային (վեբ) միջավայրում։ Web Services-ները օգտագործվում են տվյալների փոխանցման և հեռահար գործողությունների համար, և սովորաբար օգտագործում են HTTP արձանագրությունը: HTTP (Hypertext Transfer Protocol) HTTP-ն է այն արձանագրությունը, որը օգտագործվում է վեբ բրաուզերների և սերվերների միջև տվյալների փոխանցման համար։ Այն հիմնականում օգտագործվում է վեբ էջեր փոխանցելու համար, բայց նաև լայնորեն կիրառվում է API-ների և վեբ ծառայությունների համար։ HTTP Մեթոդներ 1. GET - Տվյալների ստացում սերվերից։ 2. POST - Տվյալների ուղարկում սերվերին։Նոր ռեսուրսի ստեղծում 3. PUT - Հստակ ռեսուրսի փոփոխում կամ ստեղծում սերվերի վրա։ Հստակ ռեսուրսի ամբողջական թարմացում կամ ստեղծում։ 4. DELETE - Հստակ ռեսուրսի ջնջում սերվերի վրա։ 5. PATCH - Ռեսուրսի մասամբ փոփոխում։
