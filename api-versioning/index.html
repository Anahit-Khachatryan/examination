API versioning - API տարբերակավորումը անհրաժեշտ է, որպեսզի հնարավոր լինի կատարել փոփոխություններ API-ում, առանց արդեն գոյություն ունեցող ինտեգրումներն ընդհատելու: Այն օգնում է պահպանել հետադարձ համատեղելիություն և ապահովում է, որ տարբեր տարբերակների օգտագործողները կարողանան շարունակել օգտագործել API-ն, ինչպես նաև կարող են տեղափոխվել նոր տարբերակին ժամանակի ընթացքում: Versioning` 1.
