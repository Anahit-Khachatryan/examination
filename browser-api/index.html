Browser API 1. DOM API - Տրամադրում է մեթոդներ և հատկություններ, որոնք թույլ են տալիս ծրագրավորողներին ինտերակտիվորեն փոխել և կառավարել վեբ փաստաթղթի կառուցվածքը:(getElementById, ...) 2. Fetch API - Թույլ է տալիս հեշտ և հարմար ձևով իրականացնել HTTP հարցումներ: 3. Geolocation API - Թույլ է տալիս ստանալ օգտատիրոջ աշխարհագրական դիրքը (լայնություն և երկայնություն): 4. Web Storage API - Թույլ է տալիս տվյալների տեղական պահպանումը բրաուզերի միջավայրում՝ localStorage և sessionStorage օգտագործելով: 5. Canvas API - Թույլ է տալիս նկարել գրաֆիկա և նկարներ տարրի վրա: 6. WebSocket API - Թույլ է տալիս իրական ժամանակի երկկողմանի հաղորդակցություն՝ օգտագործելով WebSockets 7. Notification API 8. setTimeout - բրաուզերի կողմից տրամադրվող գործառույթ է, որը օգտագործվում է JavaScript-ում՝ որոշակի կոդի կամ ֆունկցիայի կատարումը որոշակի ժամանակով ուշացնելու համար: 9. Eventhandleing - addEventListener Բրաուզերի API-ները ապահովում են JavaScript-ին միջերեսներ և գործիքներ, որոնք թույլ են տալիս ինտերակտիվորեն և դինամիկ կերպով աշխատել վեբ էջերի հետ: Այս API-ները JavaScript ծրագրավորողներին հնարավորություն են տալիս օգտատիրոջ հետ կապ հաստատել, բովանդակություն ցուցադրել և մանիպուլյացիայի ենթարկել, տվյալներ պահել և ստանալ, և օգտագործել սարքավորումների հնարավորությունները:
